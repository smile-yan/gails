# Dialogs Example

This example is a comprehensive example of using dialogs in Gails.

## Running the example

To run the example, simply run the following command:

```bash
go run main.go
```

## Building the example

To build the example in debug mode, simply run the following command:

```bash
gails3 task build
```

To build the example to use application icons, simply run the following command:

```bash
gails3 task package
```
